# Asclepius
Skin cancer detection app for early detection. Submission for Dicoding's _Belajar Penerapan Machine Learning untuk Android_ Class.

## App Features
- Upload Image
- Crop Image
- Analyze Image with TensorFlow Lite model

## App Architecture
- Min SDK: API 24
- Compile SDK: 34
- Tested with: AVD Pixel 6 API 30
